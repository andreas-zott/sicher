Anleitung: Webseite in Safari öffnen und als PDF versenden (iPad)

1. Webseite mit Safari auf dem iPad öffnen

  • 🔵Safari öffnen: Tippe auf das Safari-Symbol auf dem Home-Bildschirm deines iPads.
  • 🌐Webadresse eingeben: Tippe oben in die Adressleiste und gib ein:
    https://andreas-zott.github.io/sicher/
    Tippe dann auf „Los“.
  • 📄Webseite wird geladen: Die Seite öffnet sich.
    Wähle die passende Checkliste für deine iPad-Version aus.

2. Teilen-Menü öffnen

  • 🔗Tippe oben rechts auf das Teilen-Symbol (ein Quadrat mit einem Pfeil nach oben).

3. In PDF umwandeln

  • 🖨️Tippe im Teilen-Menü auf „Drucken“.
  • ⚙️Tippe oben auf „Optionen“.
    Stelle sicher, dass „PDF“ ausgewählt ist (nicht „Web-Archiv“). Tippe auf „Fertig“.
  • 📎Es erscheint eine Vorschau mit dem PDF, z. B. „Arbeitssicherheit-2025.pdf“.
  • 🖨️Es erscheint eine Druckvorschau, es werden nur Drucker angezeigt die im selben Wifi Netz eingebunden sind“.

4. PDF per E-Mail versenden

  • ⚙️Tippe oben auf „Optionen“.
    Stelle sicher, dass „PDF“ ausgewählt ist (nicht „Web-Archiv“). Tippe auf „Fertig“.
  • 📧Tippe im Teilen-Menü auf „Mail“.
  • 📝Eine neue E-Mail öffnet sich mit dem PDF als Anhang.
    Gib Empfänger, Betreff und Nachricht ein.
  • 📨Tippe oben rechts auf das Senden-Symbol (Pfeil), um die Mail zu verschicken.
